# Hardware

## Main Build

- Raspberry Pi Zero 2 W
- Waveshare 2.13" V3 e-paper display
- USB GPS receiver / G-Mouse
- 32GB microSD card
- Portable USB power bank

## Interfaces

- Wi-Fi: onboard Raspberry Pi Wi-Fi (`wlan0`)
- GPS: USB, handled by `gpsd`
- Display: GPIO-mounted Waveshare e-paper display
- Power: USB battery pack

## Notes

Wardra currently uses the Pi’s internal Wi-Fi only.

No external Wi-Fi adapter is required for the current build.
